Actor Line Highlighter

This page was created by Beckham Kunda. The goal of this project is to help actors rehearse by clicking a character name and instantly highlighting that actor's lines.

The page uses HTML data attributes, CSS classes, and JavaScript DOM events. When an actor button is clicked, JavaScript removes the highlight from all other lines and adds the highlight only to the selected actor's lines without reloading the page.

The Lost Tablet

A short original scene written for this DOM assignment.

Maya: The director said the rehearsal notes were saved on this tablet.

Leo: Then we need to find it before everyone comes back from break.

Nora: I saw it on the front table right before the lights went out.

Maya: Wait, if the lights went out, someone must have moved it by mistake.

Leo: Or someone wanted us to rehearse from memory.

Nora: That would be dramatic, but not very helpful.

Maya: Look under the script binder. It has to be nearby.

Leo: Found it! It was hiding under the costume notes.

Nora: Great. Now click our names and let the rehearsal begin.

How the JavaScript Works

Each actor button has a data-actor attribute. Each line in the script also has a matching data-actor attribute. JavaScript reads the actor name from the button that was clicked, loops through all script lines, removes the highlight class from every line, and then adds the highlight class only to the lines that match the selected actor.
